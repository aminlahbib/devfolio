# Project Images

Place your project images in the following structure:

```
public/images/projects/
├── cloud-secrets-manager/
│   ├── main.jpg          (Main thumbnail - 800x600 recommended)
│   ├── 1.jpg             (Gallery image 1 - 1200x800 recommended)
│   ├── 2.jpg             (Gallery image 2 - 1200x800 recommended)
│   ├── 3.jpg             (Gallery image 3 - 1200x800 recommended)
│   └── 4.jpg             (Gallery image 4 - 1200x800 recommended)
├── supreme-bus/
│   ├── main.jpg
│   ├── 1.jpg
│   ├── 2.jpg
│   └── 3.jpg
├── real-time-ai-tts-system/
│   ├── main.jpg
│   ├── 1.jpg
│   ├── 2.jpg
│   └── 3.jpg
└── equipment-management-system/
    ├── main.jpg
    ├── 1.jpg
    ├── 2.jpg
    └── 3.jpg
```

## Image Guidelines:
- **Format**: JPG, PNG, or WebP
- **Main image**: 800x600px (4:3 aspect ratio) - used in project cards
- **Gallery images**: 1200x800px (16:9 aspect ratio) - used in project detail gallery
- **Optimization**: Compress images for web to improve loading times

## Note:
If images are not yet available, the site will show broken image placeholders. 
Replace the placeholder URLs with your actual project screenshots when ready.
